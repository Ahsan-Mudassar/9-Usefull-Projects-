# 8 Useful Projects - Beginner Friendly 🚀

Welcome to my collection of 9 practical web development projects! This repository is designed to showcase my journey learning HTML, CSS, and JavaScript, ranging from simple logic apps to complex responsive clones.

## 📂 Projects Included

1.  **Prime Video Clone** - A visually rich clone of the Prime Video landing page.
2.  **Microsoft Web Clone** - Built using **Tailwind CSS** to practice utility-first styling.
3.  **Stopwatch Tutorial** - A functional timer with Start, Stop, and Reset logic.
4.  **To-Do List App** - A task management tool using local storage concepts.
5.  **Responsive Form Designs** - Modern, mobile-friendly Login and Sign-up forms.
6.  **Calculator** - A fully functional math tool with a clean UI.
7.  **Random Quote Generator** - Fetches or displays inspirational quotes dynamically.
8.  **Sidebar Menu** - A smooth, animated navigation sidebar for web dashboards.

## 🛠️ Tech Stack Used

*   **HTML5** - Structure and Content
*   **CSS3** - Custom styling and Animations
*   **Tailwind CSS** - Modern styling for the Microsoft Clone
*   **JavaScript** - DOM Manipulation and Logic

## 🚀 How to Run
1. Clone this repository: 
   `git clone https://github.com`
2. Open any folder (e.g., `primevideo_clone`).
3. Launch the `index.html` file in your browser.
